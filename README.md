# Passeport de Compétences MSDMC · Homneo Business School

Application web de suivi des compétences pour la formation **BAC+5 Manager de la Stratégie Digitale, Marketing & Commerciale** (RNCP 41475).

🔗 **Application en ligne :** [Voir le passeport](https://[votre-username].github.io/[votre-repo]/passeport.html)

---

## Fonctionnalités

- **Apprenant** — Cocher les 35 compétences RNCP, noter son niveau (⭐⭐⭐), ajouter des preuves/commentaires
- **Formateur** — Suivre la progression de chaque apprenant, ajouter des notes
- **Admin** — Importer les apprenants via CSV/Excel, gérer les comptes

## Stack technique

- HTML / CSS / JavaScript (single-file, aucune dépendance build)
- [Supabase](https://supabase.com) — Auth + base de données PostgreSQL
- Déployé via GitHub Pages
